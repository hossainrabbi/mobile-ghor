## Mobile Shop application

#### [Live Link](https://mobile-ghor.netlify.app/)

-   Users can add products to the cart.
-   Users can view product descriptions.
-   Users can remove added data from the cart.

### Technology Used:

-   React with Hooks.
-   Redux with Hooks.
-   Styled-component.

### upComing future
